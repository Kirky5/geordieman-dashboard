## What's Tracked

- **Overview** — sessions, calories, HR zones, weekly volume
- **VO₂ Max** — trend, classification, path to Superior
- **Running** — pace trends, cadence, stride analysis
- **Cycling** — Zwift power data, NP trends
- **Sleep** — 12 months of sleep score, duration, bedtime patterns
- **Race Predictor** — Garmin predicted race times
- **Geordieman 70.3** — race-specific readiness, 24-week training plan, countdown
- **Insights** — key observations and recommendations
