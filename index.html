What is Aerobic Efficiency?

📈 The metric

EF = Average speed (m/min) ÷ Average HR (bpm). Computed for every session over 55 minutes — long enough that aerobic effort dominates. Trending UP across weeks = aerobic engine improving (more pace at same heartbeat).

🎯 What "good" looks like

For 70.3 readiness: EF should trend up across the build. A ≥3% rise vs prior 4-week avg = meaningful aerobic gain. Plateau = need more base volume. Drop = recovery / overreaching flag.

⚠️ Decoupling caveat

True intra-session decoupling (HR drift in second half vs first half) needs lap-level data. Currently using session-level EF as a proxy — strong indicator, but not the full picture. Export Garmin lap data to upgrade this view to true decoupling %.
